# Insurance Portfolio Risk & Profitability EDA

## Objective

Exploratory Data Analysis of an 18-month motor insurance portfolio to:

- quantify loss ratio (Claims/Premium) by Province, VehicleType, Gender
- detect outliers & temporal trends
- identify high-risk vehicle makes/models

## Folder Structure

```
├── data/               # raw & processed datasets
├── notebooks/          # Jupyter notebooks
├── scripts/            # re-usable Python modules
├── .github/workflows/  # CI/CD pipelines
└── README.md
```

## Tech Stack

Python 3.11 | pandas | seaborn | matplotlib | Jupyter | GitHub Actions
